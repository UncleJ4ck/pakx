```
PPPP    A   K   K X   X 
P   P  A A  K  K   X X  
PPPP  AAAAA KKK     X   
P     A   A K  K   X X  
P     A   A K   K X   X 
```

Tiny, pipe-friendly **coreutils-style** CLI for packing / unpacking / endianness
/ byte handling.


## Build & install

```bash
cargo build
cargo install --path .
```

or 

```bash
cargo install pakx
```
